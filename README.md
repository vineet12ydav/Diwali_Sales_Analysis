# Diwali Sales Analysis

## 📊 Project Overview
This project analyzes Diwali sales data to understand customer behavior and sales trends.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 🔍 Key Insights
- IT, Aviation, and Healthcare sectors have high purchase counts
- Most customers belong to age group 26–35
- Female customers contributed more to sales

## 📁 Files
- Diwali_Sales_Analysis.ipynb

## 🚀 Conclusion
This project helps businesses understand customer trends during festive seasons and improve marketing strategies.
